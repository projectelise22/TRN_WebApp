Guides on how to use

⭐️ Creating a Project ⭐️

  1. Go to main page and check for "Projects" section.
  2. Click on "Add a new project".
  3. A new window should appear.
  4. Write your project name and add a description.
  5. Click "Create" button.

⭐️ Adding testcases ⭐️

  1. Select a project
  2. In the left side of the page, under "Project Outline" section, click for "Add testcase".
